# 2048 Game
A classic 2048 game built with JavaScript.
This project demonstrates game logic implementation, keyboard controls, score tracking, and dynamic UI updates.

## Live Preview
[View Live Demo](https://Vladislav-Korsun.github.io/js-2048-game/)


## Technologies Used
- HTML5
- SCSS (Sass)
- JavaScript (ES6)
- Object-Oriented Programming (OOP)
- BEM (Block Element Modifier)
- Gulp
- Git & GitHub
- GitHub Pages (deployment)

## Getting Started
Follow these steps to run the project locally:

### 1. Clone the repository
git clone https://github.com/Vladislav-Korsun/js-2048-game.git

### 2. Navigate to the project folder
cd js-2048-game

### 3. Install dependencies
npm install

### 4. Run the project
npm start

## Features
- Classic 2048 gameplay mechanics
- Full game logic implementation
- Dynamic board rendering
- Score tracking system
- Keyboard controls (Arrow keys)
- Win condition detection (2048 tile)
- Lose condition detection (no available moves)
- Restart game functionality
- Random tile generation (2 and 4)
- Tile merging algorithm
- Board state management
- Clean and modular code structure
- OOP-based architecture
